TikTok URL verification (one-time)

1. TikTok Developer Portal -> your app -> "URL properties" (top of app page)
2. Choose "URL prefix" (NOT domain)
3. Enter exactly:
   https://iammoeezkhann.github.io/youtube-bot/
4. Click Verify -> TikTok downloads a signature file (e.g. tiktokXXXX.txt)
5. Put that file in this docs/ folder (same level as privacy.html)
6. Push to GitHub, wait 1-2 min
7. Open https://iammoeezkhann.github.io/youtube-bot/YOUR_FILENAME.txt in browser to confirm it loads
8. Click Verify again in TikTok

Do NOT verify privacy.html or terms.html individually.
Verify the URL prefix once; that covers both pages.
